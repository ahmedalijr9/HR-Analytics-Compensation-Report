\# HR Analytics \& Compensation Report



\## 📊 Project Overview



This project presents an interactive HR Analytics Dashboard developed in Microsoft Excel to analyze workforce demographics, compensation structures, employee tenure, and payroll distribution. The dashboard transforms raw HR data into actionable insights that support workforce planning and compensation management.



\---



\## 🎯 Business Objective



Organizations need clear visibility into employee demographics and compensation trends to make informed HR decisions. This dashboard was designed to provide a comprehensive view of workforce composition, salary distribution, and employee-related KPIs through dynamic and interactive reporting.
